** ChibiOS/RT port for ARM-Cortex-M4 STM32F407.                            **

** TARGET **

The demo runs on an WaveShare STM32F4 Open 407I-C board.

** The Demo **

This demo exercies an eMMC chip attached to the SDIO peripheral.

** Build Procedure **

The demo has been tested by using the arm gcc toolchain.
